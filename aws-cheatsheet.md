# ☁️ AWS Services Cheat Sheet

## Compute
| Service | Description |
|---------|-------------|
| EC2 | Virtual machines in the cloud |
| Lambda | Serverless, run code without managing servers |
| ECS | Container orchestration service |
| EKS | Managed Kubernetes service |
| Elastic Beanstalk | Deploy and scale web apps automatically |

## Storage
| Service | Description |
|---------|-------------|
| S3 | Object storage for any type of data |
| EBS | Block storage for EC2 instances |
| EFS | Managed elastic file system |
| Glacier | Low-cost archival storage |

## Database
| Service | Description |
|---------|-------------|
| RDS | Managed relational databases (MySQL, PostgreSQL, etc.) |
| DynamoDB | NoSQL key-value and document database |
| Aurora | High-performance managed relational DB |
| ElastiCache | In-memory caching (Redis/Memcached) |

## Networking
| Service | Description |
|---------|-------------|
| VPC | Isolated virtual network in AWS |
| Route 53 | DNS and domain management |
| CloudFront | Content delivery network (CDN) |
| API Gateway | Create and manage REST/WebSocket APIs |
| ELB | Elastic Load Balancer |

## Security
| Service | Description |
|---------|-------------|
| IAM | Identity and access management |
| KMS | Key management service for encryption |
| Shield | DDoS protection |
| WAF | Web application firewall |
| Secrets Manager | Store and rotate secrets securely |

## Monitoring & DevOps
| Service | Description |
|---------|-------------|
| CloudWatch | Monitoring and observability |
| CloudTrail | Track user activity and API usage |
| CodePipeline | CI/CD pipeline automation |
| CodeDeploy | Automated application deployments |
| CloudFormation | Infrastructure as Code (IaC) |

---

> 💡 Tip: Use the [AWS Free Tier](https://aws.amazon.com/free/) to explore these services at no cost.
